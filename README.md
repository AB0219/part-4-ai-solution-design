# AI Solution Design – Manufacturing Domain

# AI-Based Defect Detection System

## Project Overview

This project proposes a Computer Vision-based AI system for automated defect detection in manufacturing industries.

The system detects defects such as dents, scratches, and cracks using deep learning models and industrial camera images.


# Business Domain

Manufacturing


# Business Problem

Manual inspection systems are slow, inconsistent, and expensive.

The AI solution automates visual quality inspection to improve efficiency and reduce defective products.


# AI Task Type

Image Classification


# Recommended Model

- CNN
- ResNet50
- EfficientNet


# Expected Benefits

- Faster inspection
- Reduced labor cost
- Improved product quality
- Lower defect rates


# Technologies Used

- Python
- TensorFlow
- OpenCV
- CNN Architectures
